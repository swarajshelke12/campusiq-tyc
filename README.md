## 🚀 About The Project
**CampusIQ** is a modern, mobile-first web application specifically crafted for the **TY-C Division**. It moves beyond static timetables by offering a live, interactive, and beautifully designed dashboard to track ongoing and upcoming classes, practical batches, and breaks.

## ✨ Key Features
- ⏱️ **Live Timeline Tracking:** Features a dynamic hero section with a circular progress ring indicating exactly how much time is left for the current class. 
- 🤖 **Meet Schedula (Chat Assistant):** An integrated smart chat assistant. Just ask questions like *"What is my next class?"* or *"When is lunch?"* and get instant context-aware answers.
- 🎨 **Aesthetic & Intuitive UI:** Enjoy color-coded subjects, sleek glassmorphism effects, fluid animations, and a highly responsive tabbed layout.
- ⚡ **Zero Dependencies:** Built entirely using pure HTML, CSS, and Vanilla JavaScript for lightning-fast load times and easy hosting.

## 🛠️ Tech Stack
- **Frontend:** HTML5
- **Logic:** Claude Code/ Kimi 2.5
- **Typography/design:** Claude

## 🚀 Getting Started
Since this is a client-side only application, getting started is incredibly simple:
1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. Enjoy your live schedule!
